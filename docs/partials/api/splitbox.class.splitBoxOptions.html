splitBoxOptions
object in module splitbox

Description

配置参数

Properties

  • animateClass

    设置动画类名

  • barTemplate

    resizeBar模板，所有的resizeBar都使用该模板生成，默认为：

    <div class="split-bar">
</div>

  • barWidth

    设置所有resizeBar的宽度

  • boxTemplate

    默认的box的模板，当box没有设置template时，使用该模板，默认为：

    <div class="split-box">
</div>

  • boxs

    需要显示的boxs，类似树形结构

  • direction

    整个布局的一级box调整尺寸的方向，水平调整设置为：x，竖直调整设置为：y，默认水平

  • exportName

    设置暴露到scope上的方法的名称

  • isFlat

    声明传入的boxs参数是数组还是对象，默认是数组

  • minSize

    设置所有box默认的最小大小，可对单个box设置minSize覆盖全局设置
